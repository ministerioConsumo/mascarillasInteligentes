# Mascarilla Inteligente

Este proyecto contiene el código de la aplicación móvil `Mascarilla Inteligente`.
Se trata de una web app encapsulada con Capacitor, con React en el frontend.

## Uso

Se recomienda utilizar [Yarn](https://yarnpkg.com/) con este proyecto.

Para descargar las dependencias:

> yarn

Para arrancar la aplicación en modo desarrollo:

> yarn start

Para generar las versiones para cada plataforma móvil:

> yarn build-mobile

## Notas

Algunas imágenes y fuentes han tenido que eliminarse del repositorio por restricciones de licencias.
Se han sustituído por placeholders para que la aplicación pueda probarse correctamente.
